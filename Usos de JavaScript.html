1. Java Script e Java são a mesma coisa?
Não. Java é uma linguagem de programação orientada a objetos desenvolvida pela Sun Microsystems. Já o JavaScript é uma linguagem de script, voltada principalmente para o desenvolvimento web, que permite interatividade e dinamismo nas páginas da web.

2. Quando surgiu o Java Script?
O JavaScript foi criado em 1995.

3. Quem foi o responsável pela criação do Java Script?
O responsável pela criação do "JS" foi Brendan Eich, um desenvolvedor da Netscape Communications.

4. Qual o caminho trilhado pelo reponsável pela criação do Java Script, trata-se de alguma empresa ainda no mercado?
O JS foi criado enquanto Brendan Eich trabalhava na Netscape. Conseguiu criá-lo em apenas 10 dias, sendo possível usá-lo no navegador da Netscape. Em relação a ser uma empresa no mercado, o JS possui código aberto e é utilizado e suportado por várias empresas e comunidades de desenvolvedores.
